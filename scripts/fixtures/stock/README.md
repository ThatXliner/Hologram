# Screenshot stock photo fixtures

These local JPEGs are used only to seed deterministic demo data for
`scripts/screenshots.ts`. They replace the old generated color gradients so the
captured UI reflects a credible professional photography workflow without
requiring network access at screenshot time.

The source photos are available through Lorem Picsum and originate on Unsplash.
They are used under the [Unsplash License](https://unsplash.com/license).
